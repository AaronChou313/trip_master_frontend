# TripMaster 旅行规划大师 - 用户手册

## 🎉 欢迎使用 TripMaster！

TripMaster 是一款功能强大的旅行规划工具，帮助您轻松规划完美的旅行体验。本手册将引导您了解所有功能的使用方法。

## 🚀 快速开始

### 1. 注册账户
首次使用需要创建账户：
- 点击"注册"按钮
- 填写用户名、邮箱和密码
- 点击"注册"完成账户创建
- 注册成功后会自动登录

### 2. 登录系统
已有账户的用户：
- 输入用户名/邮箱和密码
- 点击"登录"进入系统
- 登录状态会在页面刷新后保持

## 🗺️ 地点管理使用指南

### 搜索地点
1. 进入"地点管理"页面
2. 在搜索框中输入地点名称或关键词
3. 系统会显示相关的地点搜索结果
4. 点击地图上的标记可以查看详情

### 收藏地点
- 在搜索结果中找到喜欢的地点
- 点击"添加到收藏"按钮
- 收藏的地点会保存在您的个人列表中
- 可以随时查看和管理收藏的地点

### 地图操作
- **拖拽**：按住鼠标左键拖动地图
- **缩放**：使用鼠标滚轮或 +/- 按钮
- **选择**：点击地图标记查看详细信息

## 📅 行程规划使用指南

### 创建新行程
1. 进入"行程规划"页面
2. 点击"新建行程"按钮
3. 设置行程的基本信息：
   - 行程名称
   - 开始日期和结束日期
   - 总预算

### 添加景点
1. 在行程详情页面点击"添加景点"
2. 从已收藏的地点中选择
3. 设置：
   - 预计到达时间
   - 停留时长
   - 交通方式
   - 预算金额

### 调整顺序
- 使用拖拽功能重新排列景点访问顺序
- 系统会自动计算时间安排
- 确保行程的合理性和连贯性

### 查看行程
- **日历视图**：按日期展示每日安排
- **列表视图**：详细的时间轴展示
- **统计信息**：显示总天数、景点数、预算等

## 💰 预算管理使用指南

### 预算分类
系统将预算分为四个主要类别：
- **行程费用**：门票、导游费等
- **交通费用**：机票、火车票、当地交通等
- **住宿费用**：酒店、民宿等
- **其他费用**：餐饮、购物、应急等

### 预算设置
1. 进入"预算管理"页面
2. 为每个类别设置预算上限
3. 系统会自动从行程中同步预算数据
4. 实时显示各分类的使用情况

### 预算监控
- **进度条**：直观显示预算使用进度
- **颜色提示**：
  - 绿色：预算充足
  - 黄色：接近预算上限
  - 红色：超出预算
- **饼图分析**：各类别预算占比可视化

## 📝 备忘事项使用指南

### 创建备忘录
1. 进入"备忘事项"页面
2. 点击"新建备忘录"
3. 输入备忘录标题和内容

### Markdown编辑
支持丰富的文本格式：
```
# 标题
**粗体文字**
*斜体文字*
- 列表项
[链接文字](网址)
`代码片段`
```

### 编辑功能
- **编辑模式**：编写和修改内容
- **预览模式**：实时查看渲染效果
- **工具栏**：快速插入常用格式
- **自动保存**：内容实时保存，无需担心丢失

### 管理备忘录
- 点击左侧列表选择备忘录
- 使用删除按钮移除不需要的备忘录
- 备忘录按创建时间排序

## 📊 数据管理

### 数据导出
1. 点击右上角"导出数据"按钮
2. 系统会生成包含所有数据的JSON文件
3. 文件自动下载到您的电脑
4. 导出内容包括：地点、行程、预算、备忘录

### 数据导入
1. 点击"导入数据"按钮
2. 选择之前导出的JSON文件
3. 系统会验证文件格式
4. 导入成功后页面自动刷新

### 使用场景
- **换设备**：在新设备上恢复所有数据
- **备份**：定期备份重要旅行计划
- **分享**：与朋友分享旅行规划

## 🔧 常用操作技巧

### 快捷键
- `Ctrl + S`：保存当前编辑的内容
- `Ctrl + Z`：撤销上一步操作
- `Esc`：关闭当前弹窗或对话框

### 最佳实践
1. **定期备份**：建议每周导出一次数据
2. **及时同步**：行程有变动时及时更新
3. **预算控制**：设置合理的预算上限
4. **提前规划**：出发前一周完成详细规划

### 故障排除
**问题**：无法登录
**解决**：检查网络连接，确认用户名密码正确

**问题**：数据没有保存
**解决**：检查是否已登录，刷新页面重试

**问题**：地图无法显示
**解决**：检查网络连接，确认高德地图API正常

## 🔒 安全提醒

### 账户安全
- 使用强密码（至少8位，包含数字和字母）
- 不要在公共设备上保存登录状态
- 定期更换密码

### 数据安全
- 导出的重要数据建议加密存储
- 不要将账户信息分享给他人
- 发现异常及时联系技术支持
